## Safety interlocks [SAFETY]

| Variable | Dev MUST be | Prod safe default | Notes |
|----------|-------------|-------------------|-------|
| `DRY_RUN` | `true` | `true` | `false` enables real Meta writes (requires `META_WRITE_ENABLED=true` too) |
| `META_WRITE_ENABLED` | `false` | `false` | **Never `true` without explicit sign-off.** Second lock on Meta API writes |
| `REQUIRE_HUMAN_APPROVAL` | `true` | `true` | Prevents any automated publishing |
| `MAX_AUTOMATIC_BUDGET_INCREASE_PERCENT` | `0` | `0` | `0` disables all automated budget changes |
| `MAX_DAILY_SPEND` | `1000.0` | Set conservatively | Global hard spend ceiling in the current implementation |
| `ALLOW_BLOCKED_OVERRIDE` | `false` | `false` | `true` allows bypassing quality/policy blocks (never in prod) |
